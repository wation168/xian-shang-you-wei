# 線上有位 — 開發紀錄 v2.0

## 專案簡介
台股技術分析輔助工具，輸入股票代號自動分析支撐壓力、軌道、型態、損益比。
**產品名稱：** 線上有位
**核心定位：** 股市位置導航系統，提醒風險，不是報明牌

---

## 已完成功能

### 後端（main.py）

#### 支撐壓力（6種來源，加權評分）
- [x] 回測後低點（突破後拉回不破，權重最高6）
- [x] 低點密集區（±1.5%內多個轉折低點，權重5）
- [x] 整理平台低點（連續3根K棒波動≤2%，權重4）
- [x] 爆量支撐（權重3）
- [x] 凹洞量支撐（權重2）
- [x] 均線動態支撐 MA20/MA60（權重1）
- [x] 支撐距現價 >8% 自動降級，改用均線補位

#### 軌道分析
- [x] 上升/下降/水平軌道（R²線性迴歸，只看最近120根）
- [x] 三角形型態：收斂/擴散/上升/下降三角形
- [x] 全域座標轉換（_offset 修正，避免截短陣列污染外部）
- [x] 自動軌道線已從前端移除，改由手繪工具替代

#### 反轉型態
- [x] W底（雙底）
- [x] M頭（雙頭）
- [x] 頭肩頂
- [x] 頭肩底

#### K棒型態辨識
- [x] 單根：錘頭、射擊之星、十字星、大紅棒、大黑棒
- [x] 兩根：多頭/空頭吞噬、孕線、穿刺線、烏雲蓋頂
- [x] 三根：早晨之星、黃昏之星、三紅兵、三烏鴉
- [x] 型態預警（今天出現X，明天若...將形成...）

#### 技術指標（回傳給前端）
- [x] RSI(14)
- [x] MACD(12,26,9)
- [x] KD(9,3)
- [x] 布林通道 Bollinger Bands(20,2)
- [x] OBV 能量潮

#### 訊號
- [x] 葛蘭碧買賣點（加量能過濾，減少假訊號）
- [x] 突破壓力訊號（收盤站上壓力×1.005，爆量或凹洞量）
- [x] 跌破支撐訊號（收盤跌破支撐×0.995，爆量或凹洞量）
- [x] 只回傳最近一次訊號

#### 風險與損益
- [x] 防守位：support×0.985，最小距現價2%，最大10%
- [x] 風險報酬：風險=現價-支撐（不是防守位），更貼近實際
- [x] 結論整合器：綜合位置+K棒型態+RR比，給一致結論

#### API 端點
- [x] `GET /api/chips/{stock_id}` — 三大法人+融資融券，可選近5/10/20日
- [x] `GET /api/peers/{stock_id}` — 同產業個股（TaiwanStockInfo）
- [x] `GET /api/concept/{stock_id}` — 產業鏈概念股（TaiwanStockIndustryChain）
- [x] `GET /api/top_gainers` — 當日漲幅排行（有fallback）
- [x] FinMind 全台股資料每天快取一次
