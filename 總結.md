## 付款流程（全自動，定期定額）

1. 用戶點「立即訂閱」→ 填 Email + 自設密碼
2. 後端 `/create_order_recurring` 產生綠界定期定額授權頁，密碼暫存在後端
3. 跳轉綠界信用卡授權頁（首次授權）
4. 授權成功 → `/webhook/ecpay_recurring` 自動建帳號 + 寄開通信
5. 每期自動扣款 → Webhook 自動延長到期日 + 寄續約通知信
6. 用戶取消 → 「我的」頁面「❌ 取消定期訂閱」→ `/cancel_recurring` → 打綠界 API 終止 + 寄取消通知信

### 取消訂閱 API 說明（重要）
- 使用**全方位金流**舊版 API：`https://payment.ecpay.com.tw/Cashier/CreditCardPeriodAction`
- Content-Type：`application/x-www-form-urlencoded`
- 必填：`MerchantID`、`MerchantTradeNo`、`Action=Cancel`、`TimeStamp`（Unix timestamp，3分鐘內有效）、`CheckMacValue`
- **不可使用**站內付2.0新版（`ecpayment.ecpay.com.tw`），那是 JSON+AES 加密的不同 API
