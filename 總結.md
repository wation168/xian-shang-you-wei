## ✅ v8.5 完成項目（2026-06-16）

### 多空雷達（前身是「三件套」）

**是什麼：** 四個技術指標同時亮起才標示進場，任一熄滅就等。

**四格條件：**
1. 趨勢：MA20 > MA60（短均在長均之上）
2. MACD：柱體 > 0（動能翻正）
3. 量能：近5日均量 / 20日均量 ≥ 1.2（資金流入）
4. 乖離率：MA5 乖離在 -3%~+3%（剛站上均線，未過熱）

**進場訊號：** 四格全亮 → 系統自動標示「🎯 進場訊號」
**出場訊號：** MA5 正乖離 > +3% 且收黑K → 系統自動標示「⚠️ 出場警戒」

**API 欄位名稱：** `radar`（原本叫 `three_piece`，已全部改名）
**label 對應：** 雷達全亮🔥 / 差一格⚡ / 訊號弱👀 / 雷達靜默
**程式位置：** `_do_analyze()` 函式末尾，`_cache_set` 之前
**HTML 位置：** `_build_report_html()` 裡，操作建議 section 上方

**result 新增欄位：**
```json
{
  "bias5": MA5乖離率,
  "bias20": MA20乖離率,
  "bias_entry": 是否滿足進場條件,

}
```
