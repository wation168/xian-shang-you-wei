烹飪單位轉換完全指南

烹飪時精確的測量非常重要，尤其是在烘焙中。我們的烹飪單位轉換器支援所有常見的測量單位，包括杯(cup)、湯匙(tablespoon)、茶匙(teaspoon)、毫升(ml)、盎司(oz)和克(g)。無論您使用美式或公制食譜，這個工具都能幫助您快速準確地進行轉換。

常見烹飪單位轉換

1杯(cup) = 240毫升(ml) = 8盎司(fl oz)；1湯匙(tablespoon) = 15毫升(ml) = 3茶匙(teaspoon)；1茶匙(teaspoon) = 5毫升(ml)。對於固體材料，1杯麵粉約等於120克，1杯糖約等於200克。掌握這些基本轉換將使您的烹飪和烘焙更加便利。

如何使用轉換器

只需三個簡單步驟：首先在輸入框中輸入您要轉換的數值(例如2.5)；其次從下拉選單中選擇源單位；最後選擇目標單位。轉換器會立即顯示結果和詳細的換算信息。

美式vs公制單位

美式烹飪通常使用杯、湯匙和茶匙作為測量單位，而大多數其他國家使用公制系統(毫升和克)。我們的轉換器完全支援兩種系統之間的轉換，讓您輕鬆適應任何食譜。

精確測量的重要性

特別是在烘焙中，精確的測量會直接影響最終產品的質量。使用廚房秤進行克數測量通常比杯度測量更精確，因為杯度可能因物質密度和堆積方式而有所不同。

實用提示

轉換液體時，1杯液體 = 240毫升。轉換乾成分時，重量會因密度而異，所以按克轉換時始終參考食譜的具體材料。許多專業廚師建議使用廚房秤進行最準確的結果。

常見問題

1杯等於多少毫升？
1杯(美式) = 240毫升。這是烹飪中最常用的液體轉換。對於乾成分，由於密度不同，克數會有所不同。
1湯匙和1茶匙有什麼區別？
1湯匙(tablespoon) = 3茶匙(teaspoon)。1湯匙 = 15毫升，1茶匙 = 5毫升。這是烹飪中最基本的測量關係。
盎司(oz)和毫升(ml)之間如何轉換？
1液體盎司(fl oz) = 29.5735毫升。但請注意，盎司也可以用來測量重量(質量)，1盎司(重量) ≈ 28.35克。
1杯麵粉或糖等於多少克？
1杯麵粉約120克，1杯糖約200克，1杯奶油約225克。由於不同材料的密度不同，克數會有差異。
我應該使用杯還是克進行烹飪測量？
對於最精確的結果，特別是烘焙，建議使用克(使用廚房秤)。杯度測量較不精確，因為堆積和環境因素會影響結果。
