什麼是 GPA？

GPA（Grade Point Average，平均績點）是衡量學生學業成績的標準指標。它通過將各門課程的成績轉換為績點，再乘以對應的學分時數，最後除以總學分時數計算得出。GPA 通常採用 4.0 滿分制，但也有採用 5.0 或其他標準的學校。

如何使用 GPA 計算機

只需依次輸入課程名稱（選填）、成績或績點、學分時數和您學校的 GPA 滿分標準即可。系統會自動將字母成績轉換為績點，計算每門課程的加權分數，最終得出您的累積 GPA。您可以一次添加多門課程以獲得最準確的結果。

成績到績點的轉換標準

在 4.0 滿分制中，通常的轉換標準為：A 或 A+ 為 4.0、A- 為 3.7、B+ 為 3.3、B 為 3.0、B- 為 2.7、C+ 為 2.3、C 為 2.0、C- 為 1.7、D+ 為 1.3、D 為 1.0、F 為 0.0。不同學校可能有略微不同的標準，請確認您的學校規定。

GPA 的重要性

GPA 在申請研究所、獎學金和工作時都很重要。許多企業和學術機構會以 GPA 作為評估候選人的重要標準之一。一般來說，3.5 以上的 GPA 被認為是優秀的，3.0 以上被認為是良好的。定期檢查和計算 GPA 可以幫助您了解學業進度並制定改進計劃。

加權 GPA 計算方法

累積 GPA 的計算公式為：(課程1績點×學分1 + 課程2績點×學分2 + ... + 課程n績點×學分n) ÷ (學分1 + 學分2 + ... + 學分n)。每門課程的學分時數決定了該課程在總 GPA 中的權重，學分越多，該課程對 GPA 的影響越大。

提高 GPA 的建議

要提高 GPA，可以在成績較低的科目上下功夫，參加補習或學習小組，改進學習方法和時間管理。重修不理想的課程也可能有幫助，具體取決於您學校的政策。定期使用本計算機追蹤進度，設定明確的目標，會讓您更有動力持續改進。

常見問題

字母成績如何轉換為績點？
在標準 4.0 GPA 系統中，A/A+ = 4.0、A- = 3.7、B+ = 3.3、B = 3.0、B- = 2.7、C+ = 2.3、C = 2.0、C- = 1.7、D+ = 1.3、D = 1.0、F = 0.0。不同學校可能有不同標準，請查閱您學校的官方政策。
學分時數如何影響 GPA 計算？
學分時數作為權重因素。課程學分越多，該課程的成績對累積 GPA 的影響越大。例如，4 學分的課程成績比 1 學分課程的成績對 GPA 的影響力大 4 倍。
如何計算我的累積 GPA？
將每門課程的績點乘以該課程的學分時數，將所有結果相加，再除以總學分時數。例如，兩門課程分別為 A（4.0 績點、4 學分）和 B（3.0 績點、3 學分），累積 GPA = (4.0×4 + 3.0×3) ÷ (4+3) = 3.57。
GPA 3.5 代表什麼水準？
GPA 3.5 在 4.0 滿分制中被認為是優秀成績，通常對應 A- 或 A 級別。這個成績足以申請大多數研究所課程和獲得競爭性獎學金，在就業市場上也很有競爭力。
重修課程會如何影響我的 GPA？
這取決於您學校的政策。有些學校只計算最新的成績，有些則計算所有嘗試的平均值。建議您查詢學校政策。通常重修更高分數會提升您的 GPA，但具體規則因校而異。
